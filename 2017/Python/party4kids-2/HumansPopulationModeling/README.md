Это проект разработал Артём Тартаковский.

Учителя: Николай Гуменюк и Никита Добрянский
Локация: компания IQSpace


Проект представляет собой решение классической матиматической задачи.
Задача:
Примерно в 1400 году до н.э.в страну пришло 70 человек.
На протяжении 210 лет они жили в этой стране.
А в конце этой истории из страны вышло 2,500,000 человек.
Сколько должно быть детей у одной женщины, чтобы выполнить условия задачи?

В программе можно вводить начальные данные:
1) Начальное количество людей
2) Количество лет наблюдения 
3) Количество людей, при котором наблюдения завершаются.
