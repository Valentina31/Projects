Проект представляет программу для получения фото из Dropbox и отправки её на Microsoft Emotion API,
чтобы в результате получить данные об эмоциях людей на фотографии.

Для того, чтобы работать с этим приложением вам нужно:
1) Иметь аккаунт Dropbox
2) Создать приложение, перейдя по этой ссылке
https://www.dropbox.com/developers/apps/create
Нужно выбрать: 
- Dropbox API
- Full Dropbox
И написать любое имя

3) Зайти в настройки вашего созданного приложения и найти пункт "Generated access token"
4) Нажать на него и полученную строку вставить в код программы
5) Создать в вашем хранилище Dropbox папку под названием "party4kids"
6) Загрузить туда фотографию.
7) После этого можете использовать данное приложение.

Ссылки:
