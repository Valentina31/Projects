Учителя: Игорь Козыренко и Вадим Васильев
Локация: компания Ciklum


Проект представляет собой клиент к сети Steam. Он позваляет по вашему аккаунту в Steam, узнать у кого из ваших друзей больше всего наиграно часов по игре CS:GO.
А также представить эту информацию ввиде графика.


Для того, чтобы запустить программу нужно ввести Steam API key, а также Steam username или Steam user id.

Steam API key можно получить перейдя по следующей ссылке:
steamcommunity.com/dev/apikey
