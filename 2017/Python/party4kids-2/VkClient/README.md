Это проект разработал Артём Максимов.

Учителя: Игорь Козыренко и Вадим Васильев
Локация: компания Ciklum


Проект представляет собой клиент к социальной сети Vkontacte. 
С помощью этого клиента вы можете авторизироваться и получить подробную информацию о своих друзьях.
Вам нужно иметь аккаунт в Vkontacte, чтобы воспользоваться данным приложением.
В проекте использовался VK API.

Сссылка на сам сеть:
https://vk.com/
