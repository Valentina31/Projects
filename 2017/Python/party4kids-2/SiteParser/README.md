Учителя: Игорь Козыренко и Вадим Васильев
Локация: компания Ciklum


Проект представляет собой приложения для парсинга фриланс сайтов.
В качестве примера был выбран сайт: https://www.weblancer.net/

Приложения просматривает все предложеные работы на этом сайте и сохраняет их в удобном виде в файле result.csv
В файле содержится информация о:
1) Название работы/проекта
2) Категория
3) Цена
4) Количество заявок на работу/проект
