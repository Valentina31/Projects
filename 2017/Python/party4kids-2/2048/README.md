Это проект разработал Лев Тартаковский

Учителя: Игорь Козыренко и Вадим Васильев
Локация: компания Ciklum


Проект представляет собой реализацию игры "2048" на языке Python. В игре можно менять цвет игрового поля, цифр и граней.

"2048" - очень популярная игра для смартфонов. Примеры реализации на плафторме Android:
https://play.google.com/store/apps/details?id=com.digiplex.game&hl=ru
