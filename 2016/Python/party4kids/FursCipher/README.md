C помощью этой программы вы можете зашифровать текст на английском, русском и немецком языках с помощью шифра Цезаря или Вижинера. 
Пользователю нужно выбрать способ шифрования, затем ввыбрать язык текста, 
ввести сам текст и ещё дополнительный параметр, в зависимости от выбранного варианта шифрования.
