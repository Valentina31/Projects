<p>**Локация:** коворкинг [Терминал42](https://terminal42.com.ua/)  .</p>


<p>C помощью этой программы вы можете зашифровать текст на английском, русском и немецком языках с помощью шифра Цезаря или Вижинера.</p>
<p>Пользователю нужно выбрать способ шифрования, затем ввыбрать язык текста,
ввести сам текст и ещё дополнительный параметр, в зависимости от выбранного варианта шифрования.</p>
