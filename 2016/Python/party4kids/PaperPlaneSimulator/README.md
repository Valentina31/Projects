Это проект разработал Лев Тартаковский

Учителя: Игорь Козыренко и Вадим Васильев
Локация: компания Ciklum


Проект представляет собой симуляцию полёта бумажного самолётика. 
Программа представленна в виде игры, в которой пользователю нужно попасть бумажным самолётиком в цель. 
Для этого он задаёт 3 параметра: площадь крыла, угол броска и первоначальную скорость. 
У пользователя есть 3 попытки, чтобы попасть, после этого цель меняет свое местоположение.
